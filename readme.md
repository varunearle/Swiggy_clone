# 🍔 Food Delivery App (Swiggy Clone - Multi Role)

A full-stack food delivery web application built using Python and Streamlit, inspired by platforms like Swiggy.
This project simulates a real-world system with **three interfaces**: User, Restaurant, and Admin.

---

## 🚀 Features

### 👤 User Interface

* Browse restaurants and menus
* Place food orders
* View order history
* Real-time order status (Placed / Delivered)

---

### 🍽️ Restaurant Dashboard

* Add and manage menu items
* View incoming orders
* Update order status
* Monitor restaurant activity

---

### 👑 Admin Panel

* View all restaurants
* Monitor all orders
* Track total platform activity
* Simple analytics dashboard

---

## 🛠️ Tech Stack

* **Frontend + Backend:** Streamlit
* **Language:** Python
* **Data Handling:** Pandas
* **State Management:** Streamlit Session State

## ▶️ How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/food-delivery-app.git
cd food-delivery-app
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the application:

```bash
streamlit run app.py
```

4. Open in browser:

```bash
http://localhost:8501
```
